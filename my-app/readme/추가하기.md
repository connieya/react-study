# 추가하기 : push & concat 

### concat

a의 값은 변하지 않고 a의 값에 4가 추가 된 배열이 

b에 `깊은 복사` 된다

즉, concat 은 불변 함수이다. (a의 값이 변하지 않는다.)

```javascript
const a = [1, 2, 3];
const b = a.concat(4);

console.log('a = ', a); // [1,2,3]
console.log('b = ', b); // [1,2,3,4]
```

### push

a의 값은 변하게 된다.

push 는 불변 함수가 아니다.

```javascript
const a = [1, 2, 3];
const b = a.push(4);

console.log('a = ', a); // [1,2,3,4]
console.log('b = ', b); // [4]
```
